This is a plugin to link Media Folders With Category

Required Plugin:
https://wordpress.org/plugins/wp-media-categories/
https://wordpress.org/plugins/filebird/

- folder_to_category_link.php	= MIAN FILE
- add_cat_to_photo.php		= attach category to photo
- display_page			= plugin admin page
- folder_cat_table		= displays linking table
- check_required_plugins.php	= check required plugins if not installed
- index.php 			= protect directory view
